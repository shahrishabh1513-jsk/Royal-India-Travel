## 🌟 About The Project

Gujarat & Rajasthan Travel Explorer is a modern and fully responsive travel website built using **HTML, CSS, and JavaScript**.

The project showcases popular destinations, curated travel packages, photo galleries, testimonials, and a premium booking interface with smooth animations and dynamic content generation.

Designed as a portfolio-level frontend project demonstrating UI design, layout structuring, and interactive JavaScript functionality.

---

## ✨ Key Features

🌏 Explore Popular Destinations  
🏰 Gujarat & Rajasthan Travel Packages  
🔍 Interactive Travel Search Form  
💎 Glassmorphism UI Design  
🎯 Custom Cursor Animation  
📸 Dynamic Gallery Grid  
⭐ Testimonials Section  
📱 Fully Responsive Layout  
🚀 Smooth Scroll Navigation  
⬆️ Scroll-to-Top Button
